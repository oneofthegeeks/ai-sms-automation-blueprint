# Contributing to AI SMS Automation Blueprint

Thank you for your interest in contributing! Here's how you can help:

## How to Contribute

### Reporting Bugs
- Check if the bug has already been reported in Issues
- Include your Home Assistant version and configuration details
- Provide steps to reproduce the issue
- Include relevant logs or error messages

### Suggesting Enhancements
- Open an issue with a clear description of the enhancement
- Explain why this would be useful
- Provide examples if possible

### Pull Requests
1. Fork the repository
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test thoroughly with Home Assistant
5. Commit your changes (`git commit -m 'Add amazing feature'`)
6. Push to the branch (`git push origin feature/amazing-feature`)
7. Open a Pull Request

## Code Standards
- Follow Home Assistant blueprint best practices
- Keep prompts clear and well-documented
- Update README.md with any new features
- Add examples for new use cases

## Questions?
Feel free to open an issue for any questions or discussions!
